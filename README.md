# Project Overview

Sweet Crumbs Bakery is a responsive bakery website designed to promote a local bakery and provide customers with information about its products and services.

The website has been developed using HTML5, CSS3 and JavaScript. It is designed to be visually appealing, easy to navigate and fully responsive across desktop, tablet and mobile devices.

The website allows visitors to:

- Learn about the bakery
- View available products
- Explore bakery services
- Browse the image gallery
- Submit enquiries
- Contact the bakery
